As the number of services grow and interactions between services grows, pinpointing issues becomes a big challenge. We need distributed tracing.

XI. Logs - Treat logs as event streams

What if the problem is one single instance or some other CF component?

How to you get logs today? Lots of instances, components (like the router), etc.

This is a complicated picture. The important takeaways:

  • CF aggregates logs across instances and CF components
  • Exporting to an external analysis system like Splunk, Papertrail, ELK is trivial.
